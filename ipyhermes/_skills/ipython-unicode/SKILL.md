# Unicode Symbol Helper

When the user wants to type special symbols in IPython:

- Use IPython's built-in tab-completion: type `\alpha` then Tab → α
- Common mappings: `\beta` → β, `\gamma` → γ, `\delta` → δ, `\pi` → π
- Math: `\times` → ×, `\div` → ÷, `\pm` → ±, `\infty` → ∞
- Arrows: `\to` → →, `\leftarrow` → ←, `\Rightarrow` → ⇒
- Subscripts: `\_0` → ₀, `\_1` → ₁
- Superscripts: `\^2` → ², `\^n` → ⁿ
