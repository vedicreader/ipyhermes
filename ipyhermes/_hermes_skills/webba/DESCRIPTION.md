# webba

Provides web search and page fetching capabilities.

## Available Functions

- `web_search(query)` — Search the web and return results
- `web_fetch(url)` — Fetch and extract content from a URL

## Usage

Use web search to find documentation, examples, or recent information.
Use web fetch to read specific pages or API docs.
