## What Changed in v2

1. **Async streaming via `@patch`** — `AIAgent` is extended with `astream()` using fastcore's `patch`
   decorator, following the exact same pattern Lisette uses to extend `Chat` into `AsyncChat`.
   No `ThreadPoolExecutor` bridge. No `queue.Queue`. Pure async generator piped into Rich `Live`.
2. **shortcutpy added** — macOS Shortcuts DSL exposed in the namespace and as a bundled hermes skill.
3. **`_ai_complete` simplified** — uses the patched `AIAgent.astream()` directly, not `litellm.completion`.

## The Core Innovation: Patching AIAgent for Async Streaming

### How Lisette does it (the exact pattern to follow)

Lisette's `AsyncChat` extends `Chat` with `@patch`:

```python
# lisette/core.py — the blueprint
class AsyncChat(Chat):
    async def _call(self, msg, ...):   # inner async generator
        res = await acompletion(... stream=stream ...)
        if stream:
            res = astream_with_complete(res, postproc=cite_footnote)
            async for chunk in res: yield chunk
        ...

@patch
@delegates(Chat._call)
async def __call__(self:AsyncChat, msg=None, stream=False, ...):
    result_gen = self._call(msg, ..., stream=stream)
    if stream: return result_gen           # returns async generator
    async for res in result_gen: pass
    return res
```

### How we do it with AIAgent

`AIAgent.run_conversation()` is synchronous but already supports `stream_delta_callback` — a callable fired
on every text token via `_fire_stream_delta()`. We patch `AIAgent` to expose an async generator interface
that an asyncio event loop can consume natively, with no threads.

```python
# ipyhermes/patch.py — the entire streaming patch (≈40 lines)
from fastcore.all import *
from run_agent import AIAgent
import asyncio

@patch
async def astream(self:AIAgent,
                  prompt:str,
                  sp:str=None,
                  hist:list=None) -> str:
    """Async generator that yields text deltas then returns the full response.

    Usage:
        async for chunk in agent.astream(prompt, sp=sp, hist=hist):
            buf += chunk
        response = agent.astream.value   # full text after generator exhausts
    """
    loop = asyncio.get_running_loop()
    q = asyncio.Queue()
    sentinel = object()

    def _on_delta(chunk:str):
        "Push delta onto asyncio queue from sync thread."
        loop.call_soon_threadsafe(q.put_nowait, chunk)

    def _on_done():
        loop.call_soon_threadsafe(q.put_nowait, sentinel)

    # stash original callback; restore after
    _orig_cb = self.stream_delta_callback
    self.stream_delta_callback = _on_delta

    # run_conversation in executor so asyncio loop stays free
    fut = loop.run_in_executor(None,
        lambda: self.run_conversation(
            user_message=prompt,
            system_message=sp,
            conversation_history=hist or None,
        )
    )
    fut.add_done_callback(lambda _: _on_done())

    full = []
    try:
        while True:
            item = await q.get()
            if item is sentinel: break
            full.append(item); yield item
    finally:
        self.stream_delta_callback = _orig_cb

    result = await fut
    # attach full response for callers that need the complete Dict
    astream.last_result = result
    return ''.join(full)
```

**Key design choices:**
- `loop.call_soon_threadsafe` is the correct bridge from a sync thread into the running asyncio loop.
  It is thread-safe by design. `q.put_nowait()` from a thread is NOT safe — `call_soon_threadsafe` fixes this.
- `run_in_executor(None, ...)` uses the default `ThreadPoolExecutor` managed by asyncio — no manual pool needed.
- The sentinel pattern avoids polling or timeouts.
- `_orig_cb` preserved — if hermes was already streaming somewhere else, we don't break it.
- `astream.last_result` holds the full `run_conversation` dict for callers that need tool results, token counts, etc.

## `patch.py` — Full Implementation (provided)

This is the complete file — copy verbatim:

```python
from fastcore.all import *
from run_agent import AIAgent
import asyncio

@patch
async def astream(self:AIAgent, prompt:str, sp:str=None, hist:list=None):
    "Async generator: yields text deltas, returns full response str. Access run_conversation result via astream.last_result."
    loop = asyncio.get_running_loop()
    q, sentinel = asyncio.Queue(), object()
    _orig = self.stream_delta_callback
    self.stream_delta_callback = lambda chunk: loop.call_soon_threadsafe(q.put_nowait, chunk)
    fut = loop.run_in_executor(None,
        lambda: self.run_conversation(user_message=prompt, system_message=sp,
                                      conversation_history=hist or None))
    fut.add_done_callback(lambda _: loop.call_soon_threadsafe(q.put_nowait, sentinel))
    full = []
    try:
        while True:
            item = await q.get()
            if item is sentinel: break
            full.append(item); yield item
    finally:
        self.stream_delta_callback = _orig
    astream.last_result = await fut
    return ''.join(full)
```

async def _hermes_stream(prompt:str, agent, sp:str, hist:list, code_theme:str) -> str:
    "Consume agent.astream() into Rich Live markdown."
    buf = []
    with Live(Markdown('', code_theme=code_theme), console=_console, refresh_per_second=20) as live:

### `run_prompt`

```python
async def run_prompt(self, prompt:str):
    prompt = (prompt or '').rstrip('\n')
    if not prompt.strip(): return
    is_plan = prompt.startswith('.plan ')
    if is_plan: prompt = prompt[6:].strip()
    history_line = self.current_prompt_line()
    hist, recs = self.dialog_history()
    notes = self._collect_notes(recs)
    responses = [o['response'] for o in recs]
    tools, bad = self.resolve_tools(prompt, recs, skills=self.skills,
                                    notes=notes, responses=responses)
    ns = self.shell.user_ns
    var_xml   = _format_var_xml(_var_refs(prompt, recs, skills=self.skills, notes=notes), ns)
    shell_xml = _run_shell_refs(_shell_refs(prompt, recs, skills=self.skills, notes=notes))
    full      = self._assemble_prompt(prompt, var_xml, shell_xml, bad, history_line)
    ns[LAST_PROMPT] = prompt
    sp = _build_sysp(self.system_prompt, self.skills)
    agent = self._plan if is_plan else self._exec
    try:
        text = await _hermes_stream(full, agent, sp, self._hist, self.code_theme)
    except asyncio.CancelledError:
        text = '<system>user interrupted</system>'; print('\nstopped')
    ns[LAST_RESPONSE] = text
    ng = getattr(self.shell, '_ipythonng_extension', None)
    if ng: ng._pty_output = _strip_thinking(text)
    self.log_exact_exchange(full, text)
    self.save_prompt(prompt, text, history_line)
    self._hist += [dict(role='user', content=full), dict(role='assistant', content=text)]
```

### `_ai_complete` (Alt-. keybinding)

Uses `_fast` agent (gpt-5.4-mini, no toolsets) via `astream` for lowest latency.

```python
async def _ai_complete(self, document):
    "AI inline completion. Uses _fast agent (COMPLETE_MODEL, no tools)."
    prefix, suffix = document.text_before_cursor, document.text_after_cursor
    ctx = self.code_context(self.last_prompt_line()+1, self.current_prompt_line())
    parts = []
    if ctx: parts.append(ctx)
    parts.append(f'<current-input>\n<prefix>{prefix}</prefix>')
    if suffix.strip(): parts.append(f'<suffix>{suffix}</suffix>')
    parts.append('</current-input>\nReturn ONLY the insertion text.')
    msg, buf = '\n'.join(parts), []
    sp = 'Code completion for IPython. Return ONLY insertion text. No fences, no explanation.'
    async for chunk in self._fast.astream(msg, sp=sp): buf.append(chunk)
    return ''.join(buf).strip()
```

def _build_sysp(base:str, skills:list) -> str:

# shortcutpy

Build macOS Shortcuts from Python source using a declarative DSL.

## Usage

```python
from shortcutpy.dsl import shortcut, ask_for_text, choose_from_menu, show_result

@shortcut(color="yellow", glyph="hand")
def my_shortcut():
    name = ask_for_text("Your name?")
    show_result(f"Hello {name}")
```

Compile and sign: `compile_shortcut('my_shortcut.py')`

## Available DSL Actions
- `ask_for_text(prompt)` — prompt user for text input
- `choose_from_menu(prompt, options)` — present a choice menu
- `show_result(value)` — display output
- `save_file(value)`, `get_files()`, `resize_image(img, width=, height=)`
- `shortcut_input()` — get Share Sheet input
- See `shortcutpy.dsl` for full catalog
```

## Quality Bar

- `chkstyle ipyhermes/` → zero violations
- Launch in < 2 seconds
- Alt-. completion returns in < 3 seconds
- All 8 key bindings work
- Docker approval gate fires on every `terminal` / `execute_code` call
- karma auto-index does not delay prompt (daemon thread)
- Session save/load round-trips without data loss
- `astream` sentinel pattern: no race condition between done callback and queue drain
- `_orig_cb` restored in `finally` block even on `CancelledError`
